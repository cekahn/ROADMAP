# ROADMAP

The ROADMAP Ontology of AI Datasets, Models and Projects (ROADMAP) provides a
standardized language for the metadata that describe artificial intelligence
resources. 

ROADMAP builds on "model cards" and "datasheets for datasets" to provide a structured
taxonomy. Concepts include:
* preferred name
* synonyms and abbreviations
* Spanish and Chinese translations
* definition
* authoritative reference(s)
